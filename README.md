# Логистика 2026 — панель загрузки и стендап

Набор внутренних инструментов планирования (статичные одностраничные HTML, без сборки).

## Состав

| Файл | Что это | Открывается |
|------|---------|-------------|
| `index.html` | **Панель загрузки и сроков** — тепловая карта загрузки, детализация по проектам/исполнителям, блок «Ресурсы к планированию», фильтры, дрилл-дауны, ссылки в Jira/CAB. Читает данные из Google Sheets (CSV-публикация). | главная страница сайта |
| `standup.html` | **Стендап-доска команды** — ежедневный план в свободной форме, статусы задач, архив по дням, перенос невыполненных. | `/standup.html` |
| `apps-script/SmartsheetSync.gs` | Google Apps Script: односторонняя синхронизация Google Sheets → Smartsheet (кнопка в меню таблицы). Ставится **в Google-таблицу**, в репозитории лежит как исходник/бэкап. | — |

## Источник данных

- **Дашборд** читает опубликованный CSV из Google Sheets (лист «Логистика 2026»). Колонки определяются по именам, не по позициям — добавление/перестановка колонок не ломает дашборд.
- **Синк** Google → Smartsheet выполняется кнопкой «📤 Smartsheet» в самой Google-таблице.

## Локальный просмотр

```bash
# из корня репозитория
python3 -m http.server 8000
# открыть http://localhost:8000  (дашборд)  и  http://localhost:8000/standup.html
```
> Открывать через `http://localhost`, а не двойным кликом по файлу (`file://` блокирует загрузку CSV из-за CORS).

## Деплой на Netlify (автодеплой из Git)

1. Залить репозиторий на GitHub (см. ниже).
2. На netlify.com → **Add new site → Import from Git → GitHub** → выбрать этот репозиторий.
3. Build command — пусто; Publish directory — `.` (корень). Deploy.
4. Дальше каждый `git push` в `main` **деплоится автоматически**.

## Первая заливка на GitHub

```bash
cd путь/к/папке/репозитория
git init
git add .
git commit -m "Initial: дашборд загрузки + стендап-доска"
git branch -M main
git remote add origin https://github.com/<логин>/<репозиторий>.git
git push -u origin main
```

## Дальнейшая работа

```bash
# внёс правки в index.html или standup.html
git add .
git commit -m "что изменил"
git push           # Netlify задеплоит сам
```

## Заметки

- Дата в логике дашборда и стендапа берётся из окружения (текущая).
- Стендап-доска в этой версии хранит данные во встроенном хранилище страницы (прототип). Для общего командного доступа с разных устройств планируется перевод на Google Sheets как базу (двусторонняя связь через Apps Script Web App).
- Файлы самодостаточны: весь CSS/JS внутри каждого HTML, внешних зависимостей нет.
